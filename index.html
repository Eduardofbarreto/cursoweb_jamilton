Sobre CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML. Com CSS, podemos controlar cores, fontes, espaçamento, layout e muito mais!

Este parágrafo tem uma classe "destaque" para aplicarmos estilos específicos nele.
